# Environment Setup
Basic comands to make a new environment for evaluations

```bash
mamba create --name CADBench python==3.11 cadquery pythonocc-core uv
```

Then install requitrements using uv:

```bash
uv pip install -r requirements.txt
```

# Running Benchmarks
Two implemented scripts in this repo.
## From Pre-Generated Jsonl File
Use the `scripts/eval_jsonl.py`. See help for details and below is example usage:

```bash
python -m scripts.eval_jsonl ./example/generated.jsonl --split bench0 --num-workers 32 --variable-name result
```

## From VLM OpenAI API
You can run a vllm server (or use a foundation model API) to run a benchmark directly and save results.
Use the `scripts/eval_vlm.py`. See help for details.

Assuming you have an instance of a vllm server of `Qwen/Qwen3.5-9B` running on port 8000 locally:


```bash
python -m scripts.eval_vlm --split bench0 --model Qwen/Qwen3.5-9B --inference-num-workers 32 --render-backend blender --render-style mesh --temperature 0.2 --disable-thinking 
```
